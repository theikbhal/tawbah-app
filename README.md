# Tawbah (توبة)

**12,000 Istighfar a Day — The Sunnah of Abu Huraira (RA)**

Tawbah is a premium spiritual tracking application designed to help you return to Allah through the consistent practice of Istighfar. Inspired by the legacy of Abu Huraira (RA), who performed 12,000 Istighfar daily, this app provides a path for incremental spiritual growth.

## Features

- **Daily Tracking**: Focus on the 12k daily target.
- **Incremental Goals**: Start at 3,000/day and reach 12,000/day by the end of the year.
- **Multi-User Profiles**: Track progress for the whole family (Wife, Son, Daughter).
- **Spiritual Reports**: Weekly and monthly reflections.
- **Educational Content**: Learn the benefits of Istighfar and stories of the companions.
- **Local First**: No cloud, no login. Your spiritual journey is private.

## Design
- Premium Glassmorphic UI.
- Emerald Green and Gold aesthetic.
- Smooth transitions and haptic feedback.

## Setup
1. Clone the repo: `git clone https://github.com/theikbhal/tawbah-app.git`
2. Install dependencies: `flutter pub get`
3. Run the app: `flutter run`
